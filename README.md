# Personal Portfolio Website

A responsive personal portfolio website built with **HTML**, **CSS**, and **JavaScript**. The website presents my background, experience, projects, technical skills, leadership activities, education, and contact information in a clean, document-inspired layout.

## Features

* Responsive design for desktop, tablet, and mobile devices
* Sticky navigation with active section highlighting
* Smooth scrolling between sections
* Scroll-based section animations
* Semantic HTML structure
* Accessible keyboard focus states
* Downloadable résumé link
* Lightweight implementation with no external frameworks

## Tech Stack

* HTML5
* CSS3
* JavaScript (ES6)
* Google Fonts
* Intersection Observer API

## Getting Started

Clone the repository:

```bash
git clone https://github.com/your-username/portfolio.git
```

Open the project by launching `index.html` in your browser.

No installation or build process is required.
